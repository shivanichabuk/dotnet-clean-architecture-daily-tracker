# dotnet-clean-architecture-daily-tracker
# Daily Tracker API 

An enterprise-grade **ASP.NET Core Web API** built using **Clean Architecture principles**.  
This project is designed to track **daily tasks, notes, and productivity** for working professionals.

The goal of this project is to demonstrate **real-world backend development**, clean code practices, and scalable architecture.

---

##  Why This Project?

- Practical day-to-day use case
- Demonstrates Clean Architecture clearly
- Focuses on backend fundamentals used in real companies
- Built incrementally with meaningful commits

---

##  Architecture Overview
This solution follows **Clean Architecture**, ensuring separation of concerns and testability.
